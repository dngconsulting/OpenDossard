A Propos

Dossardeur est une application mobile destinée à assister les sportifs (cyclistes essentiellement) dans leur recherche d'épreuves sportives. Que ce soit les courses de fédérations, les cyclosportives ou les randonnées vélo, Dossardeur répertorie les évènements et propose une vue à 3 dimensions : Quand (l'agenda), Où (la géolocalisation) et Quoi (la liste d'épreuves).
Pour peu que les organisateurs aient utilisé la plateforme Open Dossard, les résultats de certaines épreuves sont également affichées ainsi que les palmarès des coureurs.
